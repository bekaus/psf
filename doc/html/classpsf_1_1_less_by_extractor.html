Detailed Description

template<typename Element, typename Extractor>
class psf::LessByExtractor< Element, Extractor >

Compare two elements with regard to a certain aspect.

Typically, an element in a spectrum represents more than one value (such as m/z, intensity, time etc.) Use this functor to compare two elements with regard to one of these values.
