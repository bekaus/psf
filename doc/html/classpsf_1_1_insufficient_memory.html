Detailed Description

Runtime error regarding insufficient RAM.

This exception is used if:

  • an operation would need more memory than available
  • the allocation of memory failed

In this sense it is more general than a std::bad_alloc, because it is used before and after trying to allocate memory. For example, a memory estimation could be done before allocating. If the estimation predicts to much needed memory, an InsufficientMemory could be thrown.
