libpsf
libpsf is a C++ peak shape function library for mass spectrometry (MS) data analysis.
libpsf development has been conducted at the Steen & Steen and Hamprecht labs.
Development and Testing
The libpsf code is considered stable; it is actively maintained at the libpsf github site. libpsf comes with a complete set of fully automated unit and memory tests, as well as coverage analysis.
If you happen to run into trouble, please do not hesitate to contact Bernhard or Marc or to make use of the github issue tracking system.
Documentation
We make use of the Doxygen documentation system to provide source documentation and annotated examples.
Licensing and Download

The libpsf python/R evaluation and data analysis code is based on GPLv2 R packages and is as such licensed under the GPLv3. Please note that this has no implications whatsoever for the C++ library.
Current stable and development versions are available from the github repository.
